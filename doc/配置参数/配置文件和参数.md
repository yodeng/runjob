##### 配置参数说明

`runjob`的配置参数可通过命令行参数传入，也可通过配置文件读取。配置文件搜索顺序：

1. `--config` 参数指定的配置文件
2. `$HOME/.config/runjob/config.ini`
3. 安装目录下的 `config.ini`

使用 `--show-config` 可查看当前加载的所有配置文件路径和参数值。


##### 配置文件格式

配置文件为 INI 格式：

```
[args]
mode = sge
cpu = 2
memory = 4G
queue = all.q
retry = 3
strict = True
```

##### 命令行参数

所有配置均可通过命令行参数覆盖，详见各命令的 `--help` 输出：

+ `runjob` / `runflow` : job 文件格式任务调度
+ `runshell` / `runsge` : shell 脚本批量任务投递
+ `qs` : SGE/本地任务查看
+ `qslurm` : Slurm 任务查看
