#### 软件安装

`runjob`安装通过`pip`安装即可，可从[github](https://github.com/yodeng/runjob)下载源码，使用`pip install`进行安装，需要`python >= 3.7`。

```
git clone https://github.com/yodeng/runjob.git
pip install ./runjob/
```

或者：

```
git clone https://github.com/yodeng/runjob.git
cd ./runjob
pip install .
```

安装时会自动下载安装相关软件依赖包，安装后使用`runjob --version`命令查看当前版本，最新版本为`v2.12.0`。
